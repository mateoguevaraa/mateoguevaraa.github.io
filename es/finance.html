Finance CS50

Año: 2023

Finance es un proyecto desarrollado como parte del curso CS50 de Harvard, una plataforma que permite a los usuarios explorar y evaluar el mercado de valores.

Con esta aplicación web, los usuarios pueden registrarse, iniciar sesión y buscar información detallada sobre acciones. Ofrece una funcionalidad robusta, permitiendo a los usuarios rastrear acciones, consultar su historial y realizar simulaciones de compra y venta en tiempo real.

Este proyecto, construido con Flask, Python y SQLite, proporciona datos actualizados del mercado utilizando la API de IEX. Con una interfaz intuitiva y amigable, Finance brinda a los usuarios una experiencia completa para explorar el mundo financiero, obteniendo información precisa y actualizada sobre acciones específicas, precios y movimientos del mercado."
