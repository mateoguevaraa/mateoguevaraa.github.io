Mateo Guevara

GrooveFinder

Año: 2024

Como un apasionado DJ y amante de la música, decidí crear GrooveFinder para satisfacer mi propia necesidad de descubrir nuevas pistas y géneros musicales de forma rápida y sencilla. Esta aplicación web utiliza la tecnología React y la potente API de Spotify para ofrecer una experiencia de descubrimiento musical sin igual.

Con GrooveFinder, los usuarios pueden explorar una amplia variedad de géneros musicales, desde el indie hasta el hip-hop, y reproducir avances de canciones directamente en la aplicación. ¿Lo mejor de todo? Con solo un clic, pueden acceder a enlaces directos a Spotify para disfrutar de la canción completa.

GrooveFinder está diseñado para hacer que el descubrimiento musical sea más emocionante y accesible que nunca. Además, con una interfaz de usuario intuitiva y amigable, GrooveFinder es adecuado para usuarios de todas las edades y niveles de experiencia.
