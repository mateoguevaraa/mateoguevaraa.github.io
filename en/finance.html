Finance CS50

Year: 2023

Finance is a project developed as part of Harvard's CS50 course, a platform that allows users to explore and evaluate the stock market.

Through this web application, users can sign up, log in, and search detailed information about stocks. It offers robust functionality, enabling users to track stocks, check their history, and simulate real-time buying and selling transactions.

This project, built using Flask, Python, and SQLite, provides real-time market data using the IEX API. With an intuitive and user-friendly interface, Finance provides users with a complete experience to explore the financial world, accessing accurate and updated information about specific stocks, prices, and market movements.
