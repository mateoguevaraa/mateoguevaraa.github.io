Tasky App

Year: 2023

Tasky is an application designed to help developers manage and track their projects. This platform offers powerful tools to create, organize, edit, and delete tasks within each project.

The idea behind this application arose because I wanted to organize myself with the projects I was doing in the Harvard CS50 course and I didn't find any application that was specific for programmers, so I decided to create one for my final project.

It uses modern technologies like Flask, SQLAlchemy, and Jinja2 on the backend, providing a robust and secure structure for data manipulation. On the frontend, HTML, CSS, and JavaScript are employed to provide an interactive and fluid user experience.

In the registration part, the application implements stringent verification regarding what the user enters in the form. It uses a library to check if the entered email actually exists, in addition to other checks on the entered username, password, and to prevent spam and password decryption it uses Google's reCAPTCHA API.

With Tasky, developers can create individual projects, add and delete tasks, as well as mark them as completed or pending. The application is designed to be intuitive and easy to use, allowing users to focus on their tasks without complications in project management.

It's an essential tool for any developer looking to optimize their productivity, as it offers a simple way to stay organized and focused on their projects.
